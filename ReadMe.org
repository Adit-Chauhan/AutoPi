#+title: AutoPi

*AutoPi* is a driving assistance project that helps in you drive safely.


* Build Instructions
#+begin_quote
Install meson and ninja for building
#+end_quote

#+begin_src bash
meson setup build
cd build
# to complile project run
meson compile
# to run tests run
meson test
#+end_src
